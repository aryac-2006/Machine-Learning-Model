# Laptop Price Prediction

## Project Description

This project predicts laptop prices using Machine Learning.

The user enters laptop specifications through a Streamlit application, and the trained Machine Learning model predicts the estimated laptop price.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib
- Streamlit

## Machine Learning Model

Linear Regression

## Project Files

- `app.py` - Streamlit frontend application
- `model-training.py` - Model training code
- `laptop_price_model.pkl` - Trained Machine Learning model
- `laptop_price_scaler.pkl` - Feature scaling file
- `laptop_price_columns.pkl` - Encoded column names
- `requirements.txt` - Required Python libraries
- `README.md` - Project documentation
